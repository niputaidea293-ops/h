# BuscaYa (Gemini Roast Search)

Esta versión es una **página web directa** con apariencia de buscador común, pero en vez de mostrar enlaces responde con una burla generada por Gemini.

## Uso directo (sin terminal)

1. Abre el archivo `web/index.html` en Firefox o Chrome.
2. Escribe una búsqueda y pulsa **Buscar**.
3. Verás la respuesta burlona de la IA en pantalla.

## Configuración gráfica

- Pulsa el botón ⚙️ para abrir la configuración.
- Puedes cambiar:
  - **API key** de Gemini.
  - **Tono** de las burlas.

La configuración se guarda en `localStorage` del navegador.
